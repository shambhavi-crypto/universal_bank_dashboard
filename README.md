# 🏦 Universal Bank Personal Loan Analysis Dashboard

Interactive dashboard for analyzing customer behavior and predicting personal loan acceptance.

## Features
- 📊 Descriptive Analysis - Demographics, financial profiles
- 🔍 Diagnostic Analysis - Comparison, correlation analysis  
- 📈 Predictive Analysis - ML model with 95%+ accuracy
- 💡 Prescriptive Analysis - Recommendations & segments
- 🎯 Customer Predictor - Individual predictions with offers

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
